Omniscience Awakened is a maintained and improved fork of the original Omniscience mod. It is designed for players who want better awareness of invisible game elements, while keeping the experience configurable and lightweight.

## Features
### See invisible entities
- Reveals invisible players, mobs, and other targeted entities
- Works through the mod’s configurable entity targeting groups
- Preserves normal invisibility behavior for untargeted entities

### Barrier block exposure
- Shows barrier blocks in survival mode
- No need to hold a barrier item

### Configurable alpha and rendering
- Adjustable opacity level for rendered invisible models
- Custom glow support for invisible entities
- Name tag visibility options with forced rendering modes

### Selective targeting
Choose which entity groups are affected:
- Players
- Monsters
- Villagers
- Animals
- Objects

Fine-grained control via the in-game config screen

### Blindness effect removal
- Optionally disable blindness fog and rendering effects
